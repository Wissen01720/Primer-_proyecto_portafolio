Proyectos

Proyecto 1

LATAMIC-SHARE

Este proyecto nace a partir de la idea de poder ayudar a difundir y apoyar a los creadores de contenido digital para evitar el opacamiento de sus obras por parte de las grandes empresas asiáticas del contenido digital, como mangas, cómics, manhwas, etc. Así, junto con un amigo, surgió la idea de crear una aplicación multiplataforma que se pueda utilizar para la publicación de todas las obras creadas por los usuarios, con un enfoque en la comunidad hispanohablante latinoamericana.

Proyecto 2

Asume el Reto por Nuestra Serranía de las Quinchas

Este proyecto nace a partir de la necesidad de ayudar a la comunidad de la Serranía de las Quinchas, en el departamento de Boyacá, Colombia, a dar a conocer este magnífico lugar turístico, con el fin de atraer turistas y generar ingresos para la comunidad. Así, en asociación entre la Universidad Santo Tomás, seccional Tunja, y el SENA, surgió la idea de crear una aplicación móvil que se pueda utilizar para dar a concer este patrimonio de la regíon

Proyecto 3

IA Security

Este proyecto nace a partir de la necesidad de ayudar a las empresas a proteger sus sistemas de información de los ciberataques. Así es como surge la idea de crear una IA privada para las empresas, con la que se puedan realizar ataques controlados a sus sistemas informáticos, detectar vulnerabilidades y, de esta manera, prevenir los ataques cibernéticos.

Proyecto 4

Aplicacion de Monitoreo de Salud

Este proyecto nace a partir de la necesidad de ayudar a las personas a monitorear su salud de manera sencilla y eficaz. Así es como surge la idea de crear una aplicación multiplataforma que se pueda utilizar para monitorear la salud de las personas, con la que se puedan realizar análisis de diferentes partes del cuerpo y tener un historial de lo que presenta el usuario, monitorear la presión arterial, el ritmo cardíaco, entre otros.

Samba

Es un software que permite a los sistemas operativos Unix compartir archivos e impresoras en una red local. Es un software libre y gratuito, y es compatible con el protocolo SMB/CIFS de Microsoft Windows.

Apache

Es un servidor web HTTP de código abierto, para plataformas Unix (BSD, GNU/Linux, etc.), Microsoft Windows, Macintosh y otras, que implementa el protocolo HTTP/1.1 y la noción de sitio virtual.
